# Endless Run

# Requirements
- `cmake` min. version 3.10
- `c++` v.17 compatible with `Visual C++ 17 (2022)` (Windows)
- `g++` min. version 9 (linux)

# Installation
```
cmake -B build
cmake --build build
```
## Dependencies
- [SFML](https://www.sfml-dev.org/)
- [JSON parser](https://github.com/nlohmann/json)
